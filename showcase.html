mill.js demonstration

mill.js uses the same syntax as the Emmet/Zen coding IDE plugin, which in turn has borrowed it from CSS selectors. This makes it very widespread, so chances are you already know how to use it. For instance, the below div was created using the command:

mill('div.square');

It also supports ids and attributes (both with value and without). Inspect the element, and check it out.

mill('div.square#user[data-user-id="1"]');

Of course there's also multiplication, using the * operator followed by a number. This will create several elements, with identical values. But what about ids? Well the way mill works is that if an element has an id but also need to be created multiple times, each element will have a different id in the format of (id)-(number).

mill('div.square#user*5');

You can use the '>' operator, which indicates a direct descendant in CSS, to nest elements.

mill('div.square>div.dot*5');

The mill method always returns an array of DOM elements, or an empty one if the command cannot be deciphered at all. Commands shouldn't contain spaces. Mill has no dependencies, and can very rapidly produce huge amounts of HTML elements.
